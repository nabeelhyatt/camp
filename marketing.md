# Melty 0.2

Chat with AI for power users

## Features

**Version control**

-   we version your chat like we version code
-   time travel
-   branch, revert, fork your chat

**Multiple models**

-   we choose the best models for each task
-   built on sonnet, haiku, flash, o1, exa
-   more coming soon

**Local filesystem**

-   Melty has access to your filesystem
-   it makes changes to local files so you have the artifacts
-   File over app
-   More abilities coming soon

**Novel UX**

-   Interruptible
-   Consult different perspectives
-   Graph view
-   Melty has access to programs just like you do
-   Keyboard shortcuts for everything
-   Powerful + super fast search

**Extensible + Configurable**

-   Melty is built on programs. Each one has a different ability
    that brings context into your chat. Examples: - Coder: edit files - Shell: run terminal commands - Web: search the internet - Summarize: summarize a new branch - CBT: cbt - Flash: quick questions - You can soon add programs yourself
-   Themes! Fonts!
